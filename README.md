# Rspec::Shiori

When you run rspec, it will cache the code and skip the test run the next time the code is the same.

## Installation

```
gem "rspec-shiori", git: 'https://github.com/longicorn/rspec-shiori.git', branch: 'master'
```

```bash
bundle install
```
